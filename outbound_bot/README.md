# 🤖 ربات فروش اوت‌باند (PHP)

ربات تلگرامی کامل برای فروش اوت‌باند/کانفیگ با تحویل **دستی**، درگاه **کارت‌به‌کارت**، و پنل مدیریت کامل داخل تلگرام.

---

## ✨ امکانات

- 🛒 خرید بر اساس **دسته‌بندی → لوکیشن → پلن**
- 📦 **تحویل دستی** اوت‌باند توسط ادمین (بعد از تایید پرداخت)
- 💳 درگاه **کارت‌به‌کارت** با ارسال رسید و تایید/رد دستی
- 👛 **کیف پول** + شارژ با رسید + تاریخچه تراکنش‌ها
- 🎁 **کد تخفیف** (درصدی یا مبلغی، با محدودیت تعداد و تاریخ انقضا)
- 👥 **زیرمجموعه‌گیری** با پاداش درصدی (قابل فعال/غیرفعال)
- 🔒 **جوین اجباری کانال** با دکمه فعال/غیرفعال‌سازی
- 👤 مدیریت کاربران (جستجو، مسدودسازی، تنظیم موجودی)
- 📢 پیام همگانی (Broadcast)
- ⚙️ تنظیم همه مقادیر از پنل ادمین: شماره کارت، صاحب کارت، پشتیبانی، کانال، حداقل شارژ، متن خوش‌آمد و ...

---

## 🛠 پیش‌نیازها (روی هاست)

- PHP نسخه **7.4 یا بالاتر**
- اکستنشن‌های فعال: `pdo_sqlite` ، `curl`
- دامنه با **SSL (https)** — تلگرام برای وبهوک https لازم دارد

---

## 🚀 مراحل نصب

### روش پیشنهادی: نصب خودکار با اسکریپت (Ubuntu 20/22/24)

یک خط زیر را روی سرور خود (با کاربر root) اجرا کنید تا منوی نصب تعاملی باز شود:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/admin6501/outtelbot/main/outbound_bot/install.sh)
```

> اگر شاخه اصلی ریپوی شما `master` است، در آدرس بالا `main` را با `master` جایگزین کنید.

اسکریپت به‌صورت خودکار:
- تمام پیش‌نیازها را نصب می‌کند (Nginx, PHP-FPM, php-sqlite3, php-curl, Certbot, Git)
- پروژه را از گیت‌هاب **کلون** می‌کند
- از شما **دامنه، توکن ربات و آیدی ادمین** را می‌پرسد
- وب‌سرور را پیکربندی و **گواهی SSL** رایگان می‌گیرد
- **وبهوک** تلگرام را ثبت می‌کند

**منوی تعاملی اسکریپت:**
```
1) 🚀 نصب / نصب مجدد        6) 📜 مشاهده لاگ‌ها
2) ▶️  استارت سرویس          7) 🔗 ثبت مجدد وبهوک
3) ⏹  استاپ سرویس           8) ⬆️  به‌روزرسانی پروژه
4) 🔄 ری‌استارت سرویس        9) 🗑  حذف کامل
5) 📊 وضعیت سرویس           0) 🚪 خروج
```

> ⚠️ پیش از نصب، رکورد **DNS (A record)** دامنه را به IP سرور اشاره دهید و پورت‌های ۸۰ و ۴۴۳ باز باشند (برای دریافت SSL).

### روش دستی (آپلود روی هاست اشتراکی)

   تمام فایل‌های پوشه `outbound_bot` را داخل یک پوشه روی هاست خود آپلود کنید
   (مثلاً `public_html/bot/`).

2. **فایل `config.php` را ویرایش کنید:**
   ```php
   define('BOT_TOKEN', 'توکن ربات از BotFather');
   $ADMIN_IDS = [123456789]; // آیدی عددی خودتان (از @userinfobot)
   ```
   > می‌توانید چند ادمین وارد کنید: `$ADMIN_IDS = [111, 222];`

3. **وبهوک را ست کنید:**
   آدرس زیر را یک‌بار در مرورگر باز کنید (مسیر را با مسیر واقعی خود جایگزین کنید):
   ```
   https://yourdomain.com/bot/set_webhook.php
   ```
   اگر پیام ✅ Success دیدید، ربات فعال است.
   > برای حذف وبهوک: `set_webhook.php?action=delete`

4. **شروع کنید:**
   - در تلگرام به ربات `/start` بدهید.
   - برای ورود به پنل مدیریت `/admin` را بزنید.

## 📋 راه‌اندازی اولیه در پنل ادمین

پس از `/admin` این مراحل را انجام دهید:

1. **⚙️ تنظیمات** → شماره کارت، نام صاحب کارت، یوزرنیم پشتیبانی را وارد کنید.
2. **🗂 دسته‌بندی‌ها** → حداقل یک دسته‌بندی بسازید (مثلاً: VIP / اقتصادی).
3. **🌍 لوکیشن‌ها** → لوکیشن‌ها را به صورت `نام | پرچم` اضافه کنید (مثلاً: `آلمان | 🇩🇪`).
4. **📦 پلن‌ها** → پلن جدید بسازید (دسته‌بندی، لوکیشن، عنوان، توضیحات، قیمت).
5. در صورت نیاز **🔒 جوین اجباری** را در تنظیمات روشن کنید و یوزرنیم کانال را وارد کنید.
   > ⚠️ ربات باید **ادمین کانال** باشد تا بتواند عضویت را بررسی کند.

## 🔄 فرایند خرید و تحویل دستی

1. کاربر پلن را انتخاب و روش پرداخت را مشخص می‌کند (کیف پول یا کارت‌به‌کارت).
2. در کارت‌به‌کارت، کاربر رسید را می‌فرستد → برای ادمین ارسال می‌شود.
3. ادمین در **🧾 سفارش‌ها** پرداخت را **تایید** می‌کند.
4. ربات از ادمین می‌خواهد **متن اوت‌باند/کانفیگ** را بفرستد.
5. ادمین متن کانفیگ را ارسال می‌کند → برای کاربر فرستاده شده و سفارش **تحویل‌شده** می‌شود.

## 🎟 فرمت ساخت کد تخفیف

در بخش «🎟 کد تخفیف» → «افزودن»:
```
کد|نوع|مقدار|حداکثر استفاده|تاریخ انقضا
```
- نوع: `percent` (درصدی) یا `amount` (مبلغی به تومان)
- حداکثر استفاده: `0` یعنی نامحدود
- تاریخ انقضا اختیاری است (`YYYY-MM-DD`)

مثال‌ها:
```
OFF20|percent|20|100
SALE|amount|15000|0|2025-12-30
```

## 📂 ساختار فایل‌ها

| فایل | توضیح |
|------|-------|
| `config.php` | توکن ربات و آیدی ادمین‌ها |
| `bot.php` | هسته: دیتابیس، توابع تلگرام، کیبوردها |
| `handlers_user.php` | منطق سمت کاربر (خرید، کیف پول، رفرال) |
| `handlers_admin.php` | پنل مدیریت کامل |
| `index.php` | نقطه ورود وبهوک |
| `set_webhook.php` | تنظیم وبهوک و ساخت دیتابیس |
| `data/bot.db` | دیتابیس SQLite (خودکار ساخته می‌شود) |

> 🔐 پوشه `data` حاوی دیتابیس است؛ یک فایل `.htaccess` برای جلوگیری از دسترسی مستقیم خودکار ساخته می‌شود.

## ❓ عیب‌یابی

- **ربات جواب نمی‌دهد:** مطمئن شوید `set_webhook.php` پیام Success داده و توکن درست است.
- **خطای دیتابیس:** پوشه‌ای که فایل‌ها در آن هستند باید قابل نوشتن باشد (`755` یا `775`).
- **جوین اجباری کار نمی‌کند:** ربات را ادمین کانال کنید و یوزرنیم را بدون `@` وارد کنید.
